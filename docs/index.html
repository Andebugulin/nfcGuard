How It Works
FOUR STEPS
TO FOCUS
01

Create Modes

Choose apps to block — or allow only essential ones. Create different modes for work, sleep, study, or any context.

02

Link NFC Tags

Register physical NFC tags as unlock keys. Place them in inconvenient locations — your kitchen, gym, or a friend's house.

03

Set Schedules

Automate blocking with per-day schedules. Different times for weekdays vs weekends. Set it and forget it.

04

Stay Focused

Blocked apps show a full-screen overlay. To unlock, physically walk to your NFC tag and tap. Friction is the feature.

Features
BUILT TO
BE UNBEATABLE

Every feature is designed around one principle: make it physically harder to give in to distraction.

⊘

Block Mode

Block specific distracting apps while everything else stays accessible. Target Instagram, TikTok, YouTube — whatever your weaknesses.

◉

Allow Mode

Nuclear option. Block everything except a whitelist of essential apps. Only phone, messages, and maps? Done.

⬡

NFC Tag Lock

Each mode can require a specific NFC tag to unlock. Wrong tag? Denied. No tag linked? Any NFC device works.

⏀

Smart Schedules

Per-day configuration with different start and end times. Auto-activate for work hours, auto-release when you're done.

↻

Persistent Service

Survives reboots, task kills, and app restarts. nfcGuard comes back every time. No escape routes.

⚿

Emergency Reset

Lost your tag? A 60-second cooldown + confirmation text safety flow lets you recover without losing your configurations.
